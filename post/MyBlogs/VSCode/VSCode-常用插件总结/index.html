VSCode-常用插件总结

Wed, Aug 21, 2019

Markdown插件

Markdown All in One

这个插件直接集成了大部分我们撰写 Markdown 时所需要辅助功能，比如进行问字加粗，斜体，表格对其等格式化，本地图片的路径自动补全等。

另外，本插件直接提供了基于 LaTeX 的数学公式渲染，支持行内数学公式，以及整段的数学公式。这样我们就不必再单独安装LaTex相关的插件了（比如 Markdown Math 插件）。

但是 Markdown All in One 并没有支持 Mermaid 渲染引擎，所以如果要在VSCode中支持流程图，时序图，甘特图的话，需要安装其它插件，比如 Markdown Preview Mermaid Support。

Markdown Preview Mermaid Support

Markdown Preview Mermaid Support 插件能够让 VS Code 的 Markdown 实时预览正确渲染 Mermaid 图。

NOTE：在本地测试，先安装了 Mermaid Preview 插件（该插件依赖于 Mermaid Markdown Syntax Highlight 插件，安装时会同时把该插件也安装上），但并不能正确渲染 Mermaid 图，后来，把这两个插件卸载，又安装了 Markdown Preview Mermaid Support 插件后，才能正常渲染 Mermaid 图。

Markdown Preview Enhanced

vscode-pdf

该插件能够在VSCode中显示PDF文档，在阅读电子书的同时做笔记非常方便。
